Assignment Requirements: Enhanced Webpage
For this assignment, you will extend your existing webpage by adding additional HTML elements and practicing basic structure.



Your webpage must include the following:



Required HTML Elements
A main heading (<h1>) with a welcome title for your webpage
A secondary heading (<h2>) introducing the content (e.g., characters, topic name)
An ordered (<ol>) or unordered (<ul>) list that displays characters from your favorite book or movie
A working hyperlink (<a>) to a related Wikipedia page
An image (<img>) related to the chosen book or movie


Proper HTML document structure, including:
<!DOCTYPE html>
<html>, <head>, and <body> tags
<meta charset="UTF-8">


File and Submission Requirements
Name your file index.html
Push the index.html file to a GitHub repository under your account
Submit the assignment by:
Uploading the index.html file and
Sharing the GitHub repository link in the submission area


Notes
Use HTML only
You may choose any book or movie
The image may be from the web or a local file in your repository
Focus on correct HTML structure and valid tags
Make sure all links and images work correctly before submitting
